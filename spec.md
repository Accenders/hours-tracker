## תיאור הפרויקט

פלטפורמה לשימוש אישי לדיווח שעות עבודה יומיות, חישוב שכר נטו/ברוטו לפי מדרגות מס ישראליות מלאות, דשבורד תובנות, היסטוריית חודשים, ובוט Claude לייעוץ פיננסי אישי.

## פונקציונליות

### 1. דיווח שעות ✅ בוצע
- כפתור כניסה מהירה / יציאה (מודד זמן בפועל)
- הוספה לפי מספר שעות ידני
- הוספה לפי שעת כניסה ושעת יציאה (מחשב אוטומטית)
- פורמט תאריך DD/MM/YYYY
- רשימת כל הדיווחים + מחיקה
- סה"כ שעות מצטבר

### 2. תמונה נוכחית (חודש נוכחי) ✅ בוצע
- סה"כ שעות שדווחו עד כה
- ברוטו מצטבר
- פירוט ניכויים: מס הכנסה, ביטוח לאומי, פנסיה, קרן השתלמות, ביטוח בריאות
- תוספות: נסיעות + ארוחות
- **נטו צפוי לקבלה**
- **עלות מעביד כוללת**

### 3. סימולטור — טרם נבנה
- הכנסת מספר שעות נוסף (היפותטי)
- תצוגת ברוטו/נטו מעודכן בהתאם

### 4. דשבורד תובנות (גרפים) — טרם נבנה
- שעות עבודה לפי שבוע/חודש
- נטו וברוטו לאורך חודשים
- פילוח ניכויים (Pie chart)
- מגמות לאורך זמן

### 5. היסטוריה — טרם נבנה
- תצוגת חודשים קודמים
- השוואה בין חודשים
- נתונים שמורים ב-Google Sheets (שורה לכל חודש)

### 6. הגדרות ✅ בוצע
- tab פרמטרים אישיים: עריכה ושמירה ל-Google Sheets
- tab מדרגות מס: טבלאות מס הכנסה וביטוח לאומי לצפייה

### 7. בוט Claude (ייעוץ אישי) — טרם נבנה
- ממשק צ'אט מובנה ב-Streamlit
- הבוט מקבל גישה לנתונים: שעות, פרמטרים, היסטוריה
- דוגמאות: "כמה שעות עוד אני צריך כדי להגיע ל-₪X נטו?", "מה ממוצע 3 חודשים אחרונים?"

## הערות לסשן הבא

### API פנימי — מפתחות dict

`get_all_hours()` מחזירה רשימת dict עם מפתחות:
- `row_index`, `date` (YYYY-MM-DD), `hours`, `notes`
- `location` — "משרד" / "בית" / "היברידי" / "—" (היעדרויות)
- `office_hours` — float, רלוונטי רק להיברידי
- `entry_type` — "עבודה" / "חופש" / "מחלה" / "חג" (ברירת מחדל: "עבודה")

`get_params()` מחזירה dict עם מפתחות בעברית:
- "תעריף שעתי", "נקודות זיכוי", "פנסיה עובד %", "פנסיה מעביד %"
- "קרן השתלמות עובד %", "קרן השתלמות מעביד %"
- "ביטוח בריאות ₪", "נסיעות ליום ₪", "ארוחות ליום ₪"

`calc_salary()` מקבלת: `hours`, `work_days` (ימי זכאות להחזרים בלבד!), `hourly_rate` + כל שאר הפרמטרים, ומחזירה dict מלא עם: `gross`, `income_tax`, `bituach_leumi`, `pension_employee`, `training_fund_employee`, `health_insurance`, `travel`, `meals`, `total_deductions`, `total_additions`, `net`, `pension_employer`, `training_fund_employer`, `employer_cost`

### כללים עסקיים חשובים
- **החזרי נסיעות/ארוחות**: רק ימי עבודה (לא היעדרויות). משרד=כן, בית=לא, היברידי=כן אם ≥4 שעות משרד
- **חופש/מחלה/חג**: כולם בתשלום מלא (שעות × תעריף). חופש+מחלה יורדים ממכסה, חג לא
- **`work_days` ב-calc_salary**: להעביר `reimbursable_days` ולא סה"כ ימי עבודה

### הפעלה
```
cd /Users/yuvalmoradov/VSCode/hours-tracker && .venv/bin/streamlit run app.py --server.headless true
```
- RTL מופעל דרך `app.py` בלבד — אין צורך לקרוא לו בכל עמוד
- keys בטופס הגדרות מתחילים ב-`s_` (s_hourly_rate, s_meals וכו') — חובה לשמור כדי למנוע קונפליקט
